This function allows to automatically create PBR materials, from texture maps exported from texturing solutions like Substance (by Allegorithmic).

Here is how it works:

    1) Choose the directory containing the texture maps. The maps should be PNG files only.

    2) The scene should contain an Occurence named OccurrenceName, on which a PBR material named MaterialName will be applied

    3) PNG maps should be named this way: OccurenceName_MaterialName_MapType.png

    4) The type of maps supported are the following ones, and they should be written exactly like this:

    • BaseColor
    • Normal
    • Metallic
    • Roughness
    • AO
